# Criando-Dashboard-de-vendas-XBOX-em-excel
Criação de dashboard de vendas em Excel para XBOX com visualização por período.
Este projeto foi desenvolvido com o objetivo de transformar dados em informações limpas e organizadas  (Importação e limpeza de dados), facilitando a analise de vendas por tabelas dinâmicas e suporte a novas estratégias(KPIs) e gráficos. 

Para armazenamento e consulta de dados foi utilizado MySQL, tabela em excel a qual seus dados foram filtrados para utilização na dashboard.

Planilha com informações sobre planos e assinaturas do Xbox, foi utilizada para responder questões:

Qual foi o faturamento total das vendas de planos anuais, diferenciando tipo de renovação? 
Quantas assinaturas do EA Play vendidas?
Quantas vendas ocorreram para o Minecraft Season Pass?
Quantas vendas utilizaram cupons de desconto?
