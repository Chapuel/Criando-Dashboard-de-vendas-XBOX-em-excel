# Criando-Dashboard-de-vendas-XBOX-em-excel
Criação de dashboard de vendas em Excel para XBOX com visualização por período
Ferramenta organizada com o intuito de responder às seguintes perguntas de negócio:

Qual o faturalmento total de vendas de produtos anuais (contendo todas as assinaturas agregadas)?
Qual o faturamento total de vendas de planos anuais, separado por auto renovação ou que não é por auto renovação?
Qual o total de vendas de assinaturas do EA Play?
Para responder cada uma dessas perguntas, foram usadas tabelas dinâmicas e gráficos. Foi seguida uma paleta de cores espefíca e os arquivos foram divididos de acordo com o método ABCDE (Assets, Base, Cálculos, Dashboard, Extras, respectivamente). No projeto final, também foram adicionadas métricas como "Receita Total", "Ticked médio", "Receita por Plano" e "Receita por Tipo de Assinatura".
